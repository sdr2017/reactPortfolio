CritterCull

An web application to help identifying those pesky critters. Users can upload images of unidentified critters (spiders are currently the only supported critter) and other users can submit species names for the critter in an attempt to identify them. Users can also search the database of images for critters that may look like one they are trying to identify.
Group Project.

Technologies Used

HTML, CSS, Bootstrap, JavaScript, jQuery, Node, MySQL, Sequelize, Express, Amazon S3

Animal Trivial

A timed trivia game about animals. The user is presented with random multiple choice questions and given 30 seconds to answer. If time runs out or the user finishes, they are taken to a results page, which shows their correct, incorrect, and unanswered totals with a corresponding graphic. After a few seconds, they are taken back for another round of questions.
Personal Project.

Technologies Used

HTML, CSS, Bootstrap, JavaScript, jQuery
